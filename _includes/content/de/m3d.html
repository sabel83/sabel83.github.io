Ein 3D-Computerspiel, das im Gymnasium, wo ich gelernt habe, stattfindet. Sie können eine meiner Klassenkameraden auswählen, um in der Schule zu wandern oder mit den Lehrern und anderen Schülern zu plaudern.

Bei dem nationalen Wettbewerb für Software-Produkte habe ich mit diesem Spiel den II. Platz erhalten. {% if include.summary == false %}

Anmerkung:

Wegen Windows-Abwärtskompatibilitäts-Problemen kann das Spiel unter Windows 2000 nicht gespielt werden. Unter Windows XP muss Windows-98-Kompatibilität eingeschaltet werden (Rechtsklick auf das Icon des Programms, Eigenschaften, Kompatibilität).

Windows 10:

Unter Windows 10 muss das Spiel manuell installiert werden, weil das Installer nicht ausgeführt werden kann. Der "Extra\Manual Setup\Moricz3D" Ordner mit allen Dateien soll vom CD zur Festplatte kopiert werden (z.B. "C:\Moricz3D") und der "DATA" Ordner mit allen Dateien soll auch vom CD in dieser Ordner kopiert werden (z.B. "C:\Moricz3D\DATA"). Danach kann das Spiel mit dem gekopierten "Moricz3D.exe" (z.B. "C:\Moricz3D\Moricz3D.exe") gestartet werden.
