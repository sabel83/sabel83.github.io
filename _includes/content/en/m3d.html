Comment:

Because of the well implemented backward compatibility of Windows this game can not be played on Windows 2000, and Windows 98 compatibility mode has to be chosen on XP (right click on the icon of the game, Properites, Compatibility tab). It works fine on Windows 98, and has never been tested on Windows 95.

Windows 10:

On Windows 10 the game needs to be installed manually, because the installer can not be executed. The "Extra\Manual Setup\Moricz3D" folder with all files in it needs to be copied from the CD to the hard drive (eg. "C:\Moricz3D") and the "DATA" folder with all files in it needs to be copied from the CD into this folder (eg. "C:\Moricz3D\DATA"). The game can be started after that with the copied "Moricz3D.exe" (eg. "C:\Moricz3D\Moricz3D.exe").
